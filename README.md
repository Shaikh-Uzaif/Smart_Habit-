# Smart Habit & Productivity Tracking System

## Overview
An Android application to track daily habits, monitor progress, and improve productivity using analytics.

## Features
- Add and manage habits
- Daily progress tracking
- Streak system
- Analytics using charts
- Reminder notifications

## Tech Stack
- Java / Kotlin
- MVVM Architecture
- SQLite (Room Database)
- MPAndroidChart
- Android Studio
